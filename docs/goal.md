# Goal

พัฒนาและปรับปรุงระบบเว็บให้ตรงตามความต้องการของผู้ใช้ โดยเน้นความเสถียร ความสวยงาม ใช้งานง่าย และไม่ทำให้ฟีเจอร์เดิมเสียหาย

## Main Objectives

- ปรับปรุง UI/UX ให้ทันสมัย สวยงาม และใช้งานง่าย
- เพิ่มหรือแก้ไขฟีเจอร์ตามคำสั่งของผู้ใช้
- รักษาโครงสร้างระบบเดิมให้ใช้งานได้
- หลีกเลี่ยงการแก้ไขส่วนที่ไม่เกี่ยวข้อง
- ตรวจสอบว่าเว็บ build และ run ได้หลังแก้ไข
