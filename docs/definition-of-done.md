# Definition of Done

งานจะถือว่าเสร็จเมื่อผ่านเงื่อนไขต่อไปนี้

- โค้ดสามารถ build หรือ run ได้โดยไม่มี error สำคัญ
- ฟีเจอร์ที่แก้ไขหรือเพิ่มสามารถใช้งานได้จริง
- ไม่มีการลบฟีเจอร์เดิมโดยไม่จำเป็น
- UI แสดงผลถูกต้องทั้ง desktop และ mobile
- ไม่มี console error ที่กระทบการใช้งานหลัก
- Firebase config, rules, และข้อมูลสำคัญไม่ถูกเปลี่ยนโดยไม่จำเป็น
- มีการตรวจสอบไฟล์ที่แก้ไขก่อนสรุปผล
