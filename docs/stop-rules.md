# Stop Rules

ให้หยุดทำงานทันทีหากพบเงื่อนไขต่อไปนี้

- ต้องลบไฟล์สำคัญจำนวนมากโดยไม่มีเหตุผลชัดเจน
- ต้องเปลี่ยน Firebase rules, auth, database structure หรือ security settings โดยไม่ได้รับอนุญาต
- ต้องใช้ secret key, API key, service account หรือข้อมูลลับที่ไม่มีใน workspace
- คำสั่งของผู้ใช้ขัดแย้งกันจนไม่สามารถตัดสินใจได้
- ไม่สามารถ build หรือ run project ได้หลังพยายามแก้ไขแล้ว
- พบว่าการแก้ไขอาจทำให้ระบบหลักเสียหาย
- ไฟล์เป้าหมายที่จำเป็นไม่มีอยู่ และไม่สามารถสร้างทดแทนได้อย่างปลอดภัย

เมื่อหยุด ให้รายงานสาเหตุอย่างชัดเจน และเสนอวิธีแก้ต่อไป
